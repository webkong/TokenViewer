## TokenViewer v0.1.0

🎉 Initial release.

### Features
- Track AI token usage across 22 providers
- Menu-bar panel with today / 7-day / 30-day / total summaries
- Cost tracking with multi-currency support
- Activity heatmap and usage trend charts
- Per-provider limits view (Codex, Copilot, Kiro)
- Supports: Claude, Codex, Kiro, Cursor, Copilot, Gemini, Opencode, Roocode, Kilo Code, Zed, Goose, Grok, Kimi, Craft, OpenClaw, Hermes, Antigravity, CodeBuddy, OhMyPi, Pi, KiloCLI, EveryCode
- Local-first: all data stored in `~/.tokenviewer/data.db`
- Chinese / English UI
