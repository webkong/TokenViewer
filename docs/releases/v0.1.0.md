## TokenViewer v0.1.0 — Initial Release 🎉

Native macOS menu-bar app that tracks AI token usage and costs across 22 providers — local-first, no cloud, no account.

### Features
- Track token usage across 22 AI coding tools (Claude, Codex, Kiro, Cursor, Copilot, Gemini, and more)
- Menu-bar panel with Today / 7-day / 30-day / Total summaries
- Cost tracking with multi-currency support
- Daily trend chart with cache hit rate
- 53-week activity heatmap
- Rate limit monitoring for Codex, Copilot, Kiro (Free/Pro/Pro+/Power), Cursor, Gemini, Kimi, Claude
- Per-model and per-provider breakdowns
- Incremental sync — fast, battery-friendly
- Chinese / English UI
- All data stored locally in `~/.tokenviewer/data.db`

### Installation
1. Download `TokenViewer.dmg`
2. Drag `TokenViewer.app` to `/Applications`
3. Launch — click the T icon in your menu bar

> First launch: go to System Settings → Privacy & Security → Open Anyway (Gatekeeper warning for ad-hoc signed app)
