## TokenViewer v0.1.0 — Initial Release 🎉

Native macOS menu-bar app that tracks AI token usage and costs across 22 providers — local-first, no cloud, no account.

### ✨ Features
- Track token usage across 22 AI coding tools (Claude, Codex, Kiro, Cursor, Copilot, Gemini, and more)
- Menu-bar panel with Today / 7-day / 30-day / Total summaries
- Cost tracking with multi-currency support (live exchange rates)
- Daily trend chart with cache hit rate and cost overlay
- 53-week activity heatmap (dashboard + panel)
- Rate limit monitoring for 8 providers: Codex, Copilot, Kiro (Free/Pro/Pro+/Power), Cursor, Gemini, Kimi, Claude, Antigravity
- Per-model and per-provider token/cost breakdowns
- Incremental sync — fast, battery-friendly (file offset cursors + rayon parallel parsing)
- Chinese / English UI, multi-currency display
- All data stored locally in `~/.tokenviewer/data.db` — no cloud, no account

### 📦 Installation
**Recommended:** Download `TokenViewer-0.1.0-Installer.pkg`
- Double-click to install → app is placed in `/Applications`
- Quarantine attribute is removed automatically — no Gatekeeper warning
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings → Privacy & Security → Open Anyway
