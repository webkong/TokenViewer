## TokenViewer v0.1.10

This release adds a custom date-range filter to the usage view, makes update detection always offer the newest version, and fixes several provider brand icons.

### ✨ Features
- Usage view gains a **Custom** time range: pick any start/end date with two date pickers, alongside the existing Today / Week / Month / All presets. Single-day windows automatically switch to the hourly trend.

### 🐛 Fixes
- Update checking now always offers the genuinely newest version. It previously trusted GitHub's "Latest" pointer, which excludes prereleases and can lag right after a publish, so an older build could be offered while a newer one already existed. The app now picks the highest published version itself.
- MiniMax, Qwen, GLM, and MiMo models showed a generic placeholder logo; each now displays its own brand icon.
- The DeepSeek icon is crisper at small sizes — the bare logomark now fills the frame instead of a small mark inside a white square.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
