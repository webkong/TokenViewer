## TokenViewer v0.1.5

This release improves limits visibility and makes provider metadata more complete.

### What's Changed

- Limits countdowns in the macOS app now display in days for longer windows, which makes weekly and monthly reset times easier to read at a glance.
- Provider metadata for limits has been expanded, improving how agent plans and windows are identified across supported tools.
- Release automation was hardened for manual workflow dispatches, reducing failure cases in GitHub-based publishing.

### Installation

**Recommended:** Download `TokenViewer-Installer.pkg`

The installer updates `/Applications/TokenViewer.app` and automatically opens the app after installation.
