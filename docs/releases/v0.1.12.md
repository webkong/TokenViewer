## TokenViewer v0.1.12

Adds MiMoCode support and improves provider display names across the dashboard.

### ✨ Improvements
- **New provider: MiMoCode** — tracks token usage from `~/.local/share/mimocode/mimocode.db`, parsing model IDs (mimo-auto, claude-sonnet-4-6, etc.), input/output/reasoning tokens, and timestamps.
- Provider display names now show proper casing (e.g., "MiMoCode" instead of "Mimocode", "KiloCLI" instead of "Kilocli") across the dashboard, model breakdown, and settings views.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
