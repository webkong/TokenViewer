## TokenViewer v0.1.4

This release improves the macOS menu-bar limits panel and stabilizes the website deployment flow.

### What's Changed

- Menu-bar limits now show every limit window for each agent, matching the dashboard limits view.
- Codex limits display both short-window and weekly usage when both are available.
- Website deployment now uses a shared GitHub Actions workflow for release and standalone website deploys.
- Website builds use the public npm registry and the stable installer download alias.

### Installation

**Recommended:** Download `TokenViewer-Installer.pkg`

The installer updates `/Applications/TokenViewer.app` and automatically opens the app after installation.
