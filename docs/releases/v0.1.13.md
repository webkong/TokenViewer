## TokenViewer v0.1.13

This release fixes repeated sync counting introduced around the v0.1.12 provider updates.

### 🐛 Fixes
- Fixed MiMoCode sync replaying already-counted rows, which could make token totals grow again on every sync even when no new MiMoCode activity happened.
- Fixed WorkBuddy quota snapshots replaying on every sync, which could add one extra conversation after each refresh even when WorkBuddy was not running.
- Sync cursor loading now uses the parser registry directly, preventing newly added providers from being missed by the cursor loader.

### ✨ Improvements
- MiMoCode parsing now follows stable row IDs and the current cache token shape, avoiding duplicated cached context in total token counts.
- Existing settings now migrate missing provider visibility entries so newly supported providers appear without manually resetting preferences.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
