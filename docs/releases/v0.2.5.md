## TokenViewer v0.2.5

This release improves the reliability of Limits refreshes and polishes menu-bar keyboard behavior.

### Fixes
- Fixed Limits refreshes so manual refreshes reliably re-run quota checks and update the UI state.
- Fixed skill linking errors to explain that a skill must be organized globally before it can be symlinked to an agent.

### Improvements
- Added Cmd+, as a global shortcut to open the Settings page from any TokenViewer page.
- Added Cmd+Q support while the menu-bar panel is focused.
- Adjusted menu-bar interaction so checking the compact status panel does not unexpectedly bring the Dashboard window forward.

### Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
