# TokenViewer 0.2.6

This release improves the Skills installer and related macOS interactions.

## What's New

- Added a Skills install flow for local folders, ZIP archives, and Git repositories.
- Supported GitHub folder URLs such as `/tree/main/skills`.
- Added multi-skill selection when a source contains more than one skill.
- Improved Git install performance with shallow clones and cache reuse between discovery and install.
- Added clearer install errors and success feedback.
- Reset the Skills install form when the sheet or page closes.
- Adjusted global toast placement for better visibility.

## Requirements

- macOS 14 or later.
