## TokenViewer v0.2.6

This release adds a Skills installer and improves the install experience for Git-based skill sources.

### 🐛 Fixes
- Fixed selected skill installs so the chosen skill IDs are correctly sent to the Rust core.
- Fixed install failures so errors are shown both in the install sheet and through global toast feedback.
- Fixed the Skills install form so it resets when the sheet or page closes.

### ✨ Improvements
- Added a Skills install flow for local folders, ZIP archives, and Git repositories.
- Added support for GitHub folder URLs such as `/tree/main/skills`.
- Added multi-skill selection when an install source contains more than one skill.
- Improved Git install performance with shallow clones and cache reuse between discovery and install.
- Adjusted global toast placement for better visibility.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
