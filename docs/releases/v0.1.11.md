## TokenViewer v0.1.11

This release fixes the menu-bar panel so it reliably refreshes when opened.

### 🐛 Fixes
- Opening the menu-bar panel now reliably triggers a sync. Previously the refresh was tied to a SwiftUI lifecycle callback that didn't fire on subsequent opens, so the panel could keep showing stale data after the first time.
- The panel-open sync is throttled (at most once per minute) so rapidly reopening the panel no longer re-parses every time. The refresh button still forces an immediate sync.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
