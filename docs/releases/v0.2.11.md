## TokenViewer v0.2.11

This release polishes the Usage dashboard and menu-bar panel after the new date-range options landed.

### 🐛 Fixes

- Fixed the menu-bar panel's range selector overflowing after Yesterday was added — the panel now shows Today / Yesterday / Week / Month / All (localized), and the Custom range stays in the full Dashboard window where its date pickers fit.
- Fixed the Dashboard range row jumping in height when toggling the Custom date pickers on and off.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
