## TokenViewer v0.2.1

This release fixes menu-bar popover dismissal, unifies agent metadata across the app, and polishes Usage and Skills controls.

### 🐛 Fixes
- Fixed the menu-bar popover sometimes staying open after focus moved away.
- Fixed popover close actions so Escape, outside clicks, and internal close requests dismiss the popover through AppKit.
- Removed the floating popover window level so `NSPopover.Behavior.transient` can close correctly on focus loss.
- Fixed Settings and Limits using different agent lists for limits visibility.

### ✨ Improvements
- Unified app-wide agent metadata through a single provider registry for Settings, About, Skills, and Limits.
- Improved the Usage custom date range control so it appears inline after Custom with compact styled date fields.
- Refined Skills row action buttons with wider button hit areas and smaller icons.
- Kept installed agents sorted first while preserving one shared source for provider names, icons, limits support, and skill participation.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
