## TokenViewer v0.2.9

This release improves Skills management and makes Git synchronization explicit and predictable.

### Skills Management
- Added global filtering and improved cross-agent organization for shared Skills.
- Reduced the cost of opening large Skill details by preloading `SKILL.md` metadata and loading additional files on demand.
- Added a confirmation flow when changing the shared Skills source directory.

### Git Sync
- Replaced merge-based synchronization with explicit Force Pull and Force Push actions.
- Force Pull replaces local Skills with the configured remote branch; Force Push replaces the remote branch with the local snapshot.
- Added destructive confirmation dialogs that explain which local or remote changes will be removed.
- Preserved filtered synchronization: selected local Skills replace their remote counterparts while unselected remote Skills remain unchanged.
- Fixed local modifications, new Skills, configured branches, and sync status reporting.

### Usage
- Added Yesterday as a quick date-range filter.

### Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
