## TokenViewer v0.1.3

This release focuses on tightening the macOS release and dashboard flow, plus a few polish fixes around limits visibility and updates.

### ✨ Features
- Settings now includes a dedicated limits visibility section for the menu-bar popover, so you can choose which agent cards appear there.
- The menu-bar popover now shows a settings shortcut in its header, making it quicker to reach configuration.
- Dashboard limits now keep showing every configured agent, with empty ones pushed to the bottom and rendered in a disabled state.

### 🐛 Fixes
- The menu-bar limits chip layout now sizes to content instead of stretching into fixed-width blocks.
- Update prompts are throttled so the install dialog only appears once per day for the same version.
- Codex incremental syncing now preserves model context across updates, preventing `unknown` labels from appearing after regular syncs.

### 📦 Installation
**Recommended:** Download `TokenViewer-Installer.pkg`
- Double-click to install → app is placed in `/Applications`
- Quarantine attribute is removed automatically — no Gatekeeper warning
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings → Privacy & Security → Open Anyway
