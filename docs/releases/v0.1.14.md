## TokenViewer v0.1.14

This release fixes Codex limits display issues and aligns the public provider count with the current MiMoCode support.

### 🐛 Fixes
- Fixed Codex short-window limit countdowns showing day-based text like `Resets in 1 day` for 5-hour windows. Short resets now display in hours or minutes.
- Fixed Codex limits parsing to include Spark quota windows from the current `additional_rate_limits` response shape.
- Fixed the website and public docs still showing 23 providers after MiMoCode support shipped.

### ✨ Improvements
- Codex limits now keep standard windows and Spark windows in a single normalized path, which keeps the main limits page and menu-bar popover consistent.
- Added a regression test covering Codex Spark limit parsing so future limits API changes are easier to catch.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
