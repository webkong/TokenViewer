## TokenViewer v0.1.15

This release fixes ZCode incremental sync losses and keeps the public app metadata in sync with the current provider set.

### 🐛 Fixes
- Fixed ZCode sync skipping rows that were still `running`, which could cause later `completed` updates to be missed.
- Fixed ZCode incremental sync for rows that share the same millisecond timestamp by adding a stable tie-breaker.
- Fixed short Codex limits windows showing day-based reset text instead of hours/minutes.
- Updated the public provider count to 24 after MiMoCode support shipped.

### ✨ Improvements
- ZCode now tracks a stable `(started_at, id)` cursor so incremental syncs can safely replay boundary rows.
- Added regression coverage for ZCode running-row replay and same-millisecond ordering.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
