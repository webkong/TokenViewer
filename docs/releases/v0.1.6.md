## TokenViewer v0.1.6

This release adds initial WorkBuddy support and tightens provider logo coverage across the macOS app.

### ✨ Features
- WorkBuddy is now supported in the macOS limits view, including local account discovery and quota window display.
- The Rust core now includes a WorkBuddy parser so local usage snapshots can be imported into TokenViewer.

### 🐛 Fixes
- Provider and model icon mapping now covers more fallback cases, so bundled logos resolve more reliably in the app.
- DeepSeek now uses the correct bundled logo instead of falling back to the letter avatar.
- WorkBuddy now uses its dedicated bundled logo instead of reusing another provider asset.

### 📦 Installation
**Recommended:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
