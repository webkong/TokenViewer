## TokenViewer v0.1.6

This release adds initial WorkBuddy support and fixes provider icon coverage in the macOS app.

### What's Changed

- Added WorkBuddy limits support in the macOS app, including local account discovery and quota window display.
- Added a WorkBuddy parser in the Rust core so local usage snapshots can be imported into TokenViewer.
- Expanded bundled provider logos and fixed provider/model icon mapping, including proper DeepSeek and WorkBuddy icons with more reliable fallback loading.

### Installation

**Recommended:** Download `TokenViewer-Installer.pkg`

The installer updates `/Applications/TokenViewer.app` and automatically opens the app after installation.
