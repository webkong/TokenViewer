## TokenViewer v0.1.7

This release makes the manual Sync action refresh both usage data and live limits together, so the menu-bar panel and dashboard stay in step after one click.

### ✨ Features
- The menu-bar Sync button now refreshes usage data and provider limits together instead of leaving limits to refresh later on their own.
- The dashboard and Settings page now use the same unified sync entry point, so every manual refresh follows one path.
- Added a small regression test to cover the unified sync flow.

### 🐛 Fixes
- Live quota cards now update immediately after a full sync, which reduces stale limits in the menu-bar panel.

### 📦 Installation
**Recommended:** Download `TokenViewer.dmg`
- Drag `TokenViewer.app` to `/Applications`
- First launch: System Settings -> Privacy & Security -> Open Anyway
- Requires macOS 14 (Sonoma) or later

**Alternative:** Download `TokenViewer-Installer.pkg`
- Double-click to install -> app is placed in `/Applications`
- The installer updates the existing app in place and opens it after installation
- Requires macOS 14 (Sonoma) or later
